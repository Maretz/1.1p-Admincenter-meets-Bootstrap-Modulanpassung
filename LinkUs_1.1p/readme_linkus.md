Tipp zur Admin Center meets Bootstrap Version

LinkUs Modul in das Menü einbinden :
Zuerst das Modul installieren.Im Anschluss unter Admin > Module die Linkus z.b. unter dem DropDown Module eintragen.
Dazu einfach das Modul wählen (linkus) , die Kategorie "Module" auswählen und den ModulNamen eintragen ( z.b. LinkUs) .
Noch die Modulrechte vergeben ( Admin, User, Rechte oder mehrauswahl möglich ) und speichern.
